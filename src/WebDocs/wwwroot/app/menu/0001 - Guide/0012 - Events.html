Events

Drapo's event system lets you attach behavior to standard DOM events using d-on-{event} attributes. When the event fires, Drapo evaluates the attribute value as a sequence of Drapo function calls separated by semicolons — no JavaScript required.

See Storage for all available functions and Data Binding for how d-model interacts with events.

Supported Event Attributes

  • d-on-click – Mouse click
  • d-on-change – Input value changed
  • d-on-keyup – Any key released
  • d-on-keyup-enter – Enter key released
  • d-on-keydown – Any key pressed
  • d-on-blur – Focus lost
  • d-on-focus – Focus gained
  • d-on-load – Element or sector loaded
  • d-on-dblclick – Double click

Basic Usage

<button d-on-click="PostData(user)">Save</button>

<input d-model="{{search.query}}"
       d-on-keyup-enter="LoadData(results)"
       placeholder="Search..." />

<select d-model="{{filter.status}}"
        d-on-change="LoadData(orders)">
    <option value="">All</option>
    <option value="open">Open</option>
</select>

Chaining Functions

Multiple Drapo functions can be executed in sequence by separating them with a semicolon. They are executed in order, left to right. See the Functions section for the full list of available functions.

<button d-on-click="AddDataItem(todos,newTodo);ClearDataField(newTodo,text)">
    Add Todo
</button>

Passing Storage Values to Functions

Storage keys and expressions can be passed as arguments to event handler functions. Drapo resolves them before calling the function.

<li d-for="item in cart">
    {{item.name}}
    <button d-on-click="RemoveDataItem(cart,item)">Remove</button>
</li>

Load Events

The d-on-load attribute fires when the element's containing sector finishes loading. This is the correct place to trigger data loading that should happen when a page or component is first displayed.

<div d-on-load="LoadData(dashboard)">
    <!-- dashboard content -->
</div>

Conditional Event Actions

You can use d-if together with event attributes to conditionally render elements that trigger events, or use the ToggleItemField / CheckDataField / UncheckDataField functions inside event handlers to implement toggle behaviour.

<button d-on-click="ToggleItemField({{item}},selected)">
    Toggle
</button>

Drag and Drop Events

Drapo provides built-in drag-and-drop support via d-dragStart and d-dragEnd attributes. These integrate with the storage system to enable dragging items between lists without custom JavaScript.

<div d-for="item in list"
     d-dragStart="true"
     d-dragEnd="true">
    {{item.name}}
</div>
