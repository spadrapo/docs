Installation

Drapo is a declarative frontend framework for .NET web applications. You can build reactive single-page applications using only HTML attributes and .NET — no JavaScript required.

Prerequisites

  • .NET 8.0 SDK or later — Download here
  • Visual Studio, VS Code, or your preferred IDE
  • Basic knowledge of HTML and C#

Step 1: Add the NuGet Package

Create a new ASP.NET Core project and add the package:

dotnet new web -n MyDrapoApp cd MyDrapoApp dotnet add package Drapo

Step 2: Configure Drapo in Program.cs

Register the Drapo service and middleware:

var builder = WebApplication.CreateBuilder(args); builder.Services.AddDrapo(); var app = builder.Build(); app.UseStaticFiles(); app.UseDrapo(options => { // Optional configuration here }); app.Run();

Step 3: Create Your First Page

Create an index.html in your wwwroot folder and include the Drapo script:

<!DOCTYPE html> <html> <head> <title>My Drapo App</title> <script src="/drapo.js"></script> </head> <body> <div d-sector="main"> <!-- Data setup --> <div d-dataKey="user" d-dataType="object" d-dataProperty-name="Jane Doe" d-dataProperty-age="30"></div> <h1>Hello, {{user.name}}!</h1> <!-- Two-way data binding --> <p>Name: <input type="text" d-model="{{user.name}}" /></p> <p>Age: <input type="number" d-model="{{user.age}}" /></p> <!-- Conditional rendering --> <p d-if="{{user.age}} >= 18">You are an adult.</p> <p d-if="{{user.age}} < 18">You are a minor.</p> <!-- Event handling --> <button d-on-click="SetDataField(user,age,30)">Set Age to 30</button> </div> </body> </html>

Run the app with dotnet run and open http://localhost:5000.

Key Concepts Used

  • d-sector — Defines a data scope for your application
  • d-dataKey — Declares a named storage item
  • d-model — Two-way data binding for inputs
  • {{expression}} — Displays data values in the template
  • d-if — Conditional rendering

Configuration Options

Drapo can be configured via DrapoMiddlewareOptions in your startup. The most common options are:

Debug Mode

app.UseDrapo(options => { if (env.IsDevelopment()) options.Debug = true; });

Caching

options.Config.UseCacheStatic = true; // Cache static files options.Config.UseCacheLocalStorage = true; // Browser local storage cache options.Config.ApplicationBuild = typeof(Program).Assembly.GetName().Version?.ToString(); // Cache busting

Components

options.Config.LoadComponents( Path.Combine(env.WebRootPath, "components"), "~/components");

Real-time (Pipes / SignalR)

options.Config.UsePipes = true;
