d-content

You can use d-content with the value internal inside a component's template to mark the element that receives the content placed between the component's tags (the component's content slot).

Example

<!-- inside the component template: mark the content slot --> <div class="card"><div d-content="internal"></div></div> <!-- usage: the inner markup is projected into the slot --> <d-card><span>Projected content</span></d-card>
