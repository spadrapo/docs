d-id

You can use d-id to give an element an identifier so it can be referenced later — typically a component or window instance, for example by ExecuteComponentFunction or ExecuteInstanceFunction. The value is mustache-resolvable, so it can be computed from data.

Example

<d-grid d-id="mainGrid"></d-grid> <input type="button" value="Refresh" d-on-click="ExecuteComponentFunction(mainGrid,Refresh)">
