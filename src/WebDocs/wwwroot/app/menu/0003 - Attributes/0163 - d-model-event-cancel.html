d-model-event-cancel

You can use d-model-event-cancel to list the DOM event(s) that cancel a pending d-model edit, reverting the control to the stored value. It is the counterpart of d-model-event, which commits the edit.

Example

Syntax:

<input d-model="{{form.name}}" d-model-event-cancel="blur" />
Stored value: {{form.name}}
