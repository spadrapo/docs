d-resize-location

You can use d-resize-location to enable drag-to-resize behavior on an element and identify the location/dimension being resized. When this attribute is present the element becomes a resize handle: pressing and dragging it resizes the target container. This is the attribute that activates the behavior; the others configure it.

Configure the behavior with:

  • d-resize-model — storage path bound to the resulting size
  • d-resize-type — resize mode (default normal)
  • d-resize-class — CSS class applied while resizing
  • d-resize-container — which ancestor is resized (levels up; default 2)
  • d-resize-preview — live preview while dragging (default false)

Syntax:

<div d-resize-location="..." d-resize-model="{{layout.width}}" d-resize-container="1" d-resize-preview="true"></div>
