d-validation

You can use d-validation to declare a validator. Declare it once (typically on a hidden element) and reference it from interactive elements (to block actions when invalid) and from display elements (to show error messages). See the Validation guide for full details and runnable examples.

Example

<div d-validation="required" style="display:none"></div>

See: [Validation guide](Validation) for complete, runnable examples.
