d-validation

You can use d-validation to attach a validator to an element by its name. Validators are defined once (with their rules) and then referenced by name from interactive elements (to block actions when invalid) and from display elements (to show error messages). See the Validation guide for how to define validators.

Example

<input type="email" d-model="{{user.email}}" d-validation="emailRequired">

See: [Validation guide](Validation) for complete, runnable examples.
