d-template

You can use d-template together with d-templateurl to inject a sector's content into a named slot of a loaded layout. The sector sets d-template to a slot name; inside the layout loaded by d-templateurl, the element marked with the same d-template name receives the sector's content. The default name is template.

Example

<!-- the sector picks a layout and the slot to fill --> <div d-sector="content" d-templateurl="~/app/parts/layout.html" d-template="body">...</div> <!-- inside layout.html, the slot element carries the matching name --> <div class="page"><div d-template="body"></div></div>
