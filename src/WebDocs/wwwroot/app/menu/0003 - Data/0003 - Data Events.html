Data Events

There is some events that you can use related to data storage


d-dataOnNotify This event will be fired every time that the storage item is notified by the observer


d-dataOnAfterLoad This event will be fired after the data has been loaded


d-dataOnAfterCached This event will be fired after the data has been cached


d-dataOnAfterContainerLoad This event will be fired after the data has been loaded from the container


d-dataOnBeforeContainerUnLoad This event will be fired before the data has been unloaded from the container
