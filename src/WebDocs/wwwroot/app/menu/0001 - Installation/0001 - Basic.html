Basic Installation

The first step is to add a reference to drapo. You can just add the package from nuget.

Your csproj will be like this below:


After that you will need to update your Startup.cs. First you need to add the service. To do that you will need to include this line of code in your ConfigureServices method. Like this:

public void ConfigureServices(IServiceCollection services) { services.AddDrapo(); }

And that's it.
