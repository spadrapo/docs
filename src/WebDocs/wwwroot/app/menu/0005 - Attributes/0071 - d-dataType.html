d-dataType

You can use d-dataType to identify the type of the storage item.

The possible values are:

  • [value](value)
  • [object](object)
  • [array](array)
  • [function](function)
  • [querystring](querystring)
  • [file](file)
  • [mapping](mapping)
  • [parent](parent)

In this case, the storage item has the name obj1 and his type is [object](object)
