d-format

Number

You can use d-format to format the ouput of your one way data binding.

Numbers can be formatted in: Decimal (D), Number (N) , Percent (P), Timespan (T)

Here a sample using timespan

Date

Dates can be formatted in: Short date pattern (d), Long date pattern (D), Short time pattern (t), Long time pattern (T), General date/time with short time (g), General date/time with long time (G), RFC1123 pattern (r), Custom (YYYY)





You can also change the [culture](d-culture) for specific values
