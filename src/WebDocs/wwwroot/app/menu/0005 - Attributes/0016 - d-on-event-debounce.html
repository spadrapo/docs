d-on-[event]-debounce

You can use d-on-[event]-debounce to debounce an event. With this attribute we will only fire the last event. The value of the attribute can be a number or true.

If the value is a number, the debounce will be that value in milliseconds. If the value is true, the debounce will be the default value of 500ms.
