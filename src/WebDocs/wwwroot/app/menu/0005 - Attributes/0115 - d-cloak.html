d-cloak

You can use d-cloak to prevent a flash of uncompiled content while Drapo is processing elements. The attribute specifies which CSS class to remove from the element after Drapo has finished processing it.

This is useful when you have elements with mustache templates or conditional content that might be visible before Drapo compiles them.

Syntax:

<div class="loading" d-cloak="loading"> <span>{{someData}}</span> </div>

The CSS class should hide the element:

.loading { display: none; }

Without d-cloak (may flash before processing):

{{cloakData.message}} - Count: {{cloakData.count}}

With d-cloak (hidden until processed):

{{cloakData.message}} - Count: {{cloakData.count}}

This content was hidden until Drapo processed it.
