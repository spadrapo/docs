d-dataLoadType

You can use d-dataLoadType to inform how the storage item will be loaded.

There is 2 options for this attribute:

  • ondemand. The storage item will be loaded when needed. Its the default value
  • startup. The storage will be loaded when the drapo reads the storage item in the render phase.

This storage item will be loaded when the storage is used. In this case, when the d-model need to fill the value inside the span

In this case, the storagem item is not used yeat. But its already loaded. You can check that using the [debugger](debugger) (ctrl + F2)

When you use the startup load type and the type of the storage item is a [function](function). The function will be executed in the render phase at the time that the storage item is loaded.
