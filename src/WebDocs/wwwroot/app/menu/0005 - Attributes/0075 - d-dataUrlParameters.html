d-dataUrlParameters

You can use d-dataUrlParameters to control how drapo will deal with parameters in a data querystring.

There is 2 options for this attribute:

  • required. This mean that all parameters in querystring will be required for the request.
  • optional. Every request will be made without check the parameters.Its the default value

Syntax:

Required
Key: {{item.Key}} Value: {{item.Value}}
Optional
Key: {{itemOptional.Key}} Value: {{itemOptional.Value}}
