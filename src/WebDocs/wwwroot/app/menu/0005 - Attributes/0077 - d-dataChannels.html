d-dataChannels

You can use d-dataChannels to sync 2 or more storagem items.

When a storage have a channel he will try to use the same value of another storage item already loaded. If it is the first one. He will be initialized following his type

Every time that you update an storage item with channel all chanels of that storage item will be updated too.

Only primitive types are supported in channels.
