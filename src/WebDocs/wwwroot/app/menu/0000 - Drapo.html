Drapo

Your frontend framework for single page applications using .NET

Drapo is a powerful, declarative frontend framework that brings reactive programming to .NET web applications without the complexity of traditional JavaScript frameworks. It uses HTML attributes to create dynamic, data-driven user interfaces with seamless server-side integration.

Key Features

  • Declarative Data Binding: Two-way data binding using simple HTML attributes like d-model
  • Template System: Dynamic content rendering with d-for loops and d-if conditions
  • AJAX Integration: Built-in data loading and posting with d-dataUrlGet and d-dataUrlSet
  • Real-time Communication: Live data updates using d-dataPipes for real-time features
  • Event Handling: Rich event system with d-on-* attributes for user interactions
  • Form Validation: Integrated validation system with custom rules and error handling
  • Component Architecture: Reusable components and sectoring for modular applications
  • Storage Management: Client-side data storage with automatic change tracking

Why Choose Drapo?

  • No JavaScript Required: Build complex SPAs using only HTML attributes and .NET
  • Seamless .NET Integration: Works naturally with ASP.NET Core and existing .NET infrastructure
  • Rapid Development: Declarative approach reduces boilerplate code significantly
  • Learning Curve: Familiar to .NET developers, no need to learn complex JavaScript frameworks
  • Performance: Optimized for .NET applications with efficient data binding and rendering

Perfect For

  • Enterprise web applications requiring rich interactivity
  • Dashboard and admin interfaces with real-time data
  • Form-heavy applications with complex validation
  • .NET teams wanting to avoid JavaScript framework complexity
  • Applications requiring tight integration with .NET backend services
