The GetExternalFrame function reads data from another frame (an iframe) on the page and stores it in a Drapo storage item. It looks up the iframe by its element id, calls the named JavaScript function inside that frame's window, and stores the value the function returns in the given data key. Use it for same-origin frames, where the frame's window function is directly accessible. For cross-origin frames use GetExternalFrameMessage, which communicates via postMessage instead.
