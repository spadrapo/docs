Reload Dynamic Pack Example



Passing true as the second argument forces a conditional request to the server using If-None-Match. If the server returns 304 Not Modified the cached pack is used; if it returns 200 the pack is refreshed with the new content.


Configuration Example

// Drapo configuration example for defining a dynamic pack:
options.Config
    .CreatePack("dynamiccomponents")
        .SetIsDynamic(true);
options.Config.HandlerPackDynamic = async (packName) => {
    // generate and return pack content at runtime
};
