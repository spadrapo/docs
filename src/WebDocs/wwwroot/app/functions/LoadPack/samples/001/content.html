Load Pack Example



Use Loaded Pack Content

After loading the pack, all components will be loaded and ready for use


Configuration Example

// Drapo configuration example for defining packs:
options.Config
    .CreatePack("uicomponents")
        .AddIncludePath("~/components/*")
        .AddExcludePath("*.ts")
        .AddExcludePath("*.d.ts")
        .AddExcludePath("*.map");
