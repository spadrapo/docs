A sample that shows how to call an Update function on specific component instances identified by their IDs


The following example shows three label input components with different IDs (I1, I2, I3). Each button calls the Update function on a specific component with a different value:

<!-- Three component instances with different IDs --> <d-labelinputinstance d-id="I1" d-caption="Name:" d-dataKeySource="userConfig" d-dataKeyField="Culture"></d-labelinputinstance> <br /> <d-labelinputinstance d-id="I2" d-caption="Name:" d-dataKeySource="userConfig" d-dataKeyField="Culture"></d-labelinputinstance> <br /> <d-labelinputinstance d-id="I3" d-caption="Name:" d-dataKeySource="userConfig" d-dataKeyField="Culture"></d-labelinputinstance> <br /> <!-- Buttons that call the Update function on specific components --> <input type="button" value="Update I1 to pt" d-on-click="ExecuteComponentFunction(I1,Update,pt)" /> <input type="button" value="Update I2 to en" d-on-click="ExecuteComponentFunction(I2,Update,en)" /> <input type="button" value="Update I3 to pt" d-on-click="ExecuteComponentFunction(I3,Update,pt)" /> </d-code>

In this example:

  • I1, I2, I3 are the component IDs (first parameter)
  • Update is the function name to call (second parameter)
  • pt, en are the values passed to the Update function (third parameter)
