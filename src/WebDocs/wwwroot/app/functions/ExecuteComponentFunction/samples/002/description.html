A sample that demonstrates calling a component function with multiple parameters and using mustache expressions


This example shows how to pass multiple parameters to a component function and use data from mustache expressions:

<div d-dataKey="componentData" d-dataValue='{"title": "Hello", "message": "World"}'></div> <!-- Component instance --> <d-messagecomponent d-id="MSG1"></d-messagecomponent> <!-- Button calling component function with multiple parameters --> <input type="button" value="Set Message" d-on-click="ExecuteComponentFunction(MSG1,SetMessage,{{componentData.title}},{{componentData.message}})" /> <!-- Button calling component function with dynamic parameters --> <input type="button" value="Update Title" d-on-click="ExecuteComponentFunction(MSG1,UpdateTitle,New Title)" /> </d-code>

In this example:

  • MSG1 is the component ID
  • SetMessage and UpdateTitle are function names
  • {{componentData.title}}, {{componentData.message}} are mustache expressions passed as parameters
  • New Title is a static text parameter

This demonstrates that ExecuteComponentFunction can handle multiple parameters of different types (mustache expressions, static text).
