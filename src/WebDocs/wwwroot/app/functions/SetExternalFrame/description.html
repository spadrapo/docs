The SetExternalFrame function sends data from a Drapo storage item into another frame (an iframe) on the page. It looks up the iframe by its element id and calls the named JavaScript function inside that frame's window, passing the storage item's data as the argument. Use it for same-origin frames, where the frame's window function is directly accessible. If the frame has not finished loading yet, the call is deferred until the frame's load event. For cross-origin frames use SetExternalFrameMessage, which communicates via postMessage instead.
