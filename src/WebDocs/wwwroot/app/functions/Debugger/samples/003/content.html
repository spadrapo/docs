Debugger Configuration Management

Available Commands:
  • reload - Reloads the debugger with current configuration from session storage
  • persist - Persists or removes debugger configuration in session storage

Usage Notes:

  • Open the debugger first by pressing Ctrl + F2
  • Configure your debugging preferences in the debugger interface
  • Use 'persist' to save configuration across page reloads
  • Use 'reload' to refresh the debugger with saved configuration
